== Description ==

Divert all of the 404 broken pages to your home page. Manage 404 errors, and improve your site. No knowledge of Apache, htaccess or Nginx required.

== Installation ==

1. Download and Upload all plugin files to your `/wp-content/plugins` folder, or else you can directly install the plugin from the WordPress plugins directory (/wp-admin/plugin-install.php).
2. Activate the plugin through the '/wp-admin/plugins.php' page from your wp admin. 
3. once activated, there is no futher actions required. all of your 404 pages will instantly redirected to home page.
